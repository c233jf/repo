# CMake

CMake 是一个开源、跨平台的构建系统。它并不会直接构建软件，而是根据配置文件可以生成各种平台、各种构建工具使用的标准构建文件，然后再用一般方式构建。这使得开发者可以使用他熟悉的方式构建他的软件。
