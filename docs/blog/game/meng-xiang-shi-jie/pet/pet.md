# 宠物篇

## 如何判断宠物是否值得培养

1. 技能数量：满技能或满技能减一，满技能推荐做 PK 宠
2. 主要 ZZ 需要红
3. 成长为 5 则合格：0 级宝宝血量除以体质（排除强壮、化境加的血量），成长影响升级增加的基础属性

**注意：** 吃元宵虽然会增加宝宝 ZZ 但是不会影响宠物的成长

::: tip

- 如果洗出了垃圾 ZZ 的多技能宝宝，可以留着合珍品宠
- 缺日常宠的话，如果洗出 ZZ 主、体、速、防 3 红甚至全红（主 ZZ 一定要是红而且不能相差满 ZZ 太多），并且是 3 技能带有特殊的，可以考虑留着做日常宠
  :::

## 推荐携带宠物

### 日常

日常推荐两只暴力单物理输出宠，一只暴力物群宠（法宠日常比较吃力，不考虑）

如果你是高等级玩家（例如 145 级 或 165 级），则推荐日常宠物带龙魂或者星辉，因为这两只宠物的输出非常超模

**加点：**

全力

**最终属性：**

记得使用属性丹去调一下宠物属性以满足下表

|      | 148 级                    | 168 级                    |
| ---- | ------------------------- | ------------------------- |
| 血量 | 4000 左右                 | 5000 左右                 |
| 攻击 | 3000 左右                 | 4000 左右                 |
| 速度 | 650 左右（全力 550 左右） | 800 左右（全力 600 左右） |

**技能：**

必带：特殊、终极闪击、终极偷袭

可选：终极必杀、终极强力、终极保命、终极命中（打 51 星兵的红狼可加伤害）、终极报复、追击

日常宠物优先输出为主，最好是四技能，如果打书的时候把特殊打掉了，可使用连击、吸血代替，如果技能格子不够可以舍弃保命，追求输出对冲

天赋技能：高级气势、高级偷袭强化，剩余一个则从剩余技能的强化中选或着选加速或加伤的

觉醒技能：聚力 > 攻坚 > 迅影

#### 子女

子女分为 4 种类型：物攻、法攻、拉血、封

不过现在的子女日常已经没有什么用处，基本只在 PK 才会使用，而且通常是作为最后一位出场

- 物攻：开天辟地、各种破\*式、吃枪奥义学追魂刺和 3 环
- 法攻：三味真火
- 拉血：吃佛奥义、通常是龟速拉、肉
- 封：现在基本没人用，所以这里不考虑

## 培养

### 如何提升宠物属性

- 给宠物装备
- 吃神仙果：每个 +1 基础属性
- 吃魔灵果：每个 +1 属性点

### 如何提升宠物 ZZ

- 吃元宵：在 30 个元宵之前推荐吃极品元宵，如果主 ZZ 已经超过了 2280，则推荐提升副 ZZ（**注意：超过 30 个元宵，效果减半，ZZ 超过 2280，效果也会减半**）
- 做图鉴任务：每提升一星级 +3 全 ZZ
- 飞升培养（吃宠）：每个 ZZ 可培养 4 次，主 ZZ 每次 +14，总共 +56，其它 ZZ +12，总共 +48
- 装备饰品：每件饰品 +20 ZZ，总共可携带 3 件，即 +60 ZZ

### 如何增加宠物潜力点

- 飞升
