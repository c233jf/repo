# 编程范式

编程范式指的是一类编程风格，包含了代码组织结构以及处理常见编程问题的方式。

编程范式和编程语言之间的关系可能十分复杂，因为一个编程语言可以支持多种编程范式，但有些编程语言可能在设计阶段就考虑了某种范式，所以会具有比其它语言更利于使用这种范式的特性。

范式之间不是冲突、相互排斥的，这意味着你可以在一次编程中同时使用多种范式。
