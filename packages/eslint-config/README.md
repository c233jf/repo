# eslint-config

Chenjf personal eslint config

## Install

With npm:

`npm i -D @chenjf/eslint-config`

With yarn:

`yarn add -D @chenjf/eslint-config`

With pnpm:

`pnpm add -D @chenjf/eslint-config`

## Configs

- base

```js
export default {
  extends: ['@chenjf'],
}
```

- vue

```js
export default {
  extends: ['@chenjf/eslint-config/vue'],
}
```
